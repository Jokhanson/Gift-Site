# 🎁 Gift Site

Персональный сайт-открытка с сюрпризом. Вы создаёте страницу с поздравлением, видео и QR-кодом, а получатель может добавить свои фото-воспоминания.

## ✨ Возможности

- **Flip-карточка** — открытка с анимацией переворота и конфетти
- **Видео** — вставка YouTube / VK Video с таймкодом
- **QR-код** — стилизованный градиентный QR с иконкой, можно скачать
- **Фото** — загрузка до 5 фото через Supabase Storage (сжатие 1200px)
- **Редактирование** — правка любой открытки
- **Счётчик просмотров** — 👁️ показывает сколько раз открывали
- **Анимированный фон** — медленно переливающийся градиент
- **Адаптивность** — работает на телефонах и планшетах

## 🛠️ Стек

- Чистый HTML / CSS / JS — никаких сборщиков
- [Supabase](https://supabase.com) — база данных + файловое хранилище
- [QRCode Generator](https://github.com/kazuhikoarase/qrcode-generator) — генерация QR
- GitHub Pages — хостинг

## 🚀 Быстрый старт

### 1. Создайте проект Supabase

Зарегистрируйтесь на [supabase.com](https://supabase.com), создайте проект и скопируйте **Project URL** и **anon key**.

### 2. Настройте базу данных

Откройте SQL Editor в Supabase и выполните содержимое [`supabase-setup.sql`](supabase-setup.sql) — это создаст таблицу `pages`, настроит RLS и storage bucket.

Дополнительно (для счётчика просмотров):
```sql
ALTER TABLE pages ADD COLUMN IF NOT EXISTS views INTEGER DEFAULT 0;
```

### 3. Подключите к сайту

Откройте [`script.js`](script.js) и замените `SUPABASE_URL` и `SUPABASE_ANON_KEY` на свои:

```js
const SUPABASE_URL = 'https://ваш-проект.supabase.co'
const SUPABASE_ANON_KEY = 'ваш-anon-ключ'
```

### 4. Опубликуйте на GitHub Pages

1. Залейте файлы в GitHub репозиторий
2. Перейдите Settings → Pages
3. Выберите ветку `main` и корневую папку `/`
4. Сайт будет доступен по адресу `https://ваш-username.github.io/Gift-Site`

## 📖 Как пользоваться

1. Нажмите **+ Новая открытка** в боковом меню
2. Заполните поля (повод, заголовок, обращение, текст, подпись, стикер)
3. Опционально добавьте ссылку на YouTube / VK видео
4. После создания откройте страницу — нажмите «Открыть» для переворота карточки
5. Нажмите **Показать QR-код**, чтобы дать ссылку получателю
6. Получатель может загрузить свои фото через интерфейс

## 🎨 Кастомизация

- **Цвета фона** — правьте градиент в `body` в `style.css`
- **Скорость анимации** — измените `20s` в `@keyframes bgShift`
- **Стикеры** — можно задать любой эмодзи при создании (до 4 символов)

## 📁 Структура

```
index.html          — вся разметка
style.css           — стили (карточка, модалки, фото, адаптив)
script.js           — роутер, Supabase API, генерация QR, конфетти
supabase-setup.sql  — DDL для Supabase
```
